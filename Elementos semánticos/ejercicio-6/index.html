Lista de Definiciones:

HTML
HyperText Markup Language. Es el lenguaje de marcado estándar para crear páginas web.
CSS
Cascading Style Sheets. Es un lenguaje de hojas de estilo utilizado para describir la presentación de un documento HTML.
JavaScript
Es un lenguaje de programación que permite crear contenido dinámico e interactivo en páginas web.
