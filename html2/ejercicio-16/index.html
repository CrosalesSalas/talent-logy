Glosario

HTML
HTML (Lenguaje de Marcado de Hipertexto) es el estándar utilizado para la creación y el diseño de páginas web.
CSS
CSS (Hojas de Estilo en Cascada) es un lenguaje de estilo utilizado para describir la presentación de un documento HTML.
JavaScript
JavaScript es un lenguaje de programación utilizado para agregar interactividad y dinamismo a las páginas web.
